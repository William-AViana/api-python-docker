new api python with flask and docker 

* O Ambiente Virtual (venv)
    Antes de instalar o Flask, criamos um Ambiente Virtual. Pensa nisto como um laboratório isolado. Se instalares algo lá dentro, não estraga o resto do teu computador.

Comando para criar: ```python -m venv venv```

Comando para ativar: ```source venv/bin/activate``` (Mac/Linux) ou ```venv\Scripts\activate``` (Windows).
